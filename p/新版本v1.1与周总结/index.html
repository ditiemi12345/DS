新版本(v1.1)与周总结

2025.08.01
阅读时长: 3 分钟
loading... 次

（本文章中，Ditiemi Studio将简写为DS）

从7月26日正式开放DS网站以来，已经过去了7天的时间！在这7天的时间里，本网站从诞生到逐渐发展，都离不开大家的支持！今天，本网站迎来了第一次小更新（v1.1）！这篇文章中，不仅会介绍本次小更新的所有内容，也会记录上本网站的周总结！

更新-v1.1

功能 内容
关于 完善该界面及其功能，”关于“界面中的文章主要包含了欢迎词、DS分支及其主要负责内容、网站的注意事项以及我想说的话等。
背景 网页背景不再黑白单调，加上了蓝紫色的点线组合，电脑端配合鼠标的移动，效果可能会更为明显。
Emoji 网页支持了全部的Emoji表情，在这次更新中，也在已发布的文章以及关于界面中加入了部分Emoji表情。
无刷新加载页面 利用PJAX技术，使您在切换页面时，不会刷新页面，导致卡顿的状况。
加载动画 在您打开网站时，如果信号卡顿或者加载缓慢，会出现白底和灰色加载动画，您只需要稍作等待即可。
Github 头像下方的一个Github图标，点击即可进入DS的Github页面。
目录 在文章主题右侧的”#目录“功能，点击其中的小标题即可跳转到小标题所对的内容。
其他 更改了页面底部的内容；更改头像下方灰字内容；部分页面显示英文已转为中文；修复文章底部概率显示代码的问题；更改文章发布日期的格式；修改了部分文章内容。
优化 优化了网页加载排版，主题与内容的速度。
删除的内容 删除了阅读时间功能，友链界面，（暂时）评论功能。

周总结（7.26-8.1）

其实这次小更新感觉也能叫大更新了哈哈哈哈，以后的小更新内容可能没有那么多，请大家见谅。

🚀 成长里程碑

  • 上周我们很荣幸收获到了150+次的访问，收到了2条用户体验反馈，感谢大家为v1.1更新指明了方向！
  • 不计本文章，本周共发表了2篇文章！分别是D-M与D-D分支的分别一篇文章！感谢大家给予我的创作力量！
  • 借助了一些插件，使界面的加载时间平均下降了30%左右（2.1s → 1.4s）！
  • 支持的Emoji表情给文章内容带来了更多的乐趣！

💡 总结&洞察&目标

  • 删除的“评论”功能因功能未完善暂时下线，预计在后续更新以更强的形态回归。
  • 删除的“阅读时间”功能因为算法不精准，可能会导致误导而暂时下线，预计在重新调整时间算法后在下个更新中回归。
  • “搜索”功能有时还会出现搜索时间慢，搜不出结果，搜索后重新加载搜索界面的问题，我们将重视此问题，在下个更新中修复！
  • 深色模式切换有时会出现切换不了，卡顿的情况、会在下个更新中改善；移动端背景动效适配也列入了开发路线图。
  • 在Beta版测试中的“文章浏览数”功能并未取得完满成功，存在一定的问题，我们还在努力完善中，发布时间未定。

您的每一次点击都意义非凡。DS 将坚持快速响应、持续优化的原则，努力为大家奉上更优质的作品！

祝您愉快，

Ditiemi Studio
