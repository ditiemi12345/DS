重大更新(v2.0)

2025.08.21
阅读时长: 3 分钟

大家久等啦！今天，DS网站迎来了一次非常重大的更新！更新内容众多，深得大家的喜爱！同时，希望今天更新的内容符合大家的胃口，给大家带来更有趣更便捷的网页体验！那么下面就来介绍介绍本次的更新吧~

#分类 与 相关文章

现在，您可以在主页右侧边栏”∞归档“功能下方发现一个新的功能：#分类。您可以在此快速查找同类型的文章（如D-M，D-S等），同时，我们删除了文章标题前的分支内容，换为每篇文章有着其对应的分类。

同时，我们在文章末尾增加了”相关文章“功能，该功能与”#分类“功能结合，在您预览完文章时，会为您推送几篇同类型的文章，我们相信这些功能的发布能够为您提供更好的使用体验。

文章内图片

我们感到非常高兴：我们可以在文章内添加图片了！在接下来的文章里，您将可以在文章内看到图片内容，我们相信能够让您更形象地理解内容。

二次元看板娘（维护时间：每月5日~12日）

二次元看板娘是本次更新地重头戏！在您进入网站时，可以看到网页左下角出现了一位二次元看板娘。您可以与其互动。

  • 看板娘-互动发言

    在您的鼠标指针移至一个链接上时，您会发现她能与相对应链接互动发言。当您的鼠标指针碰到以模型高度为边长的正方形区域时，看板娘也会与您的鼠标进行互动发言。

  • 看板娘-指针跟踪

    看板娘会的眼睛会始终跟着鼠标指针的移动而移动。当您的鼠标指针碰到以模型高度为边长的正方形区域时，能够更好地体现出该效果。

  • 看板娘-右栏

    当您的鼠标指针碰到以模型高度为边长的正方形区域时，您会发现模型右侧出现了几个功能键。

    • 右栏-语录

      当您点击第一个功能键——语录时，看板娘会说出某些富有哲理的句子。该内容为联网寻找，当您在某一个句子停留一段时间后，会显示出该内容的出处

    • 右栏-飞机游戏

      当您点击第二个功能键——飞机游戏时，您会发现网页左上方出现了一个飞机（锐角等腰三角形）您可以通过键盘上的方向键进行操控，空格为发射子弹。被子弹打到的内容会消失，您可以在右下角查看您的分数（每消失一个内容即可获得相应分数！），按Esc键可退出游戏，但如果想要继续浏览正常的网页，麻烦您刷新一下网页啦！

    • 右栏-表情

      当您点击第三个功能键——表情时，看板娘的脸部可能发生表情的微妙变化。

    • 右栏-截图

      当您点击第四个功能键——截图时，网站将会下载一个看板娘目前状态的截图。

    • 右栏-退出

      当您点击第五个功能键——退出时，看板娘将会隐藏，您可以在网页左下角的”看板娘“按钮重新呼出看板娘。

  • 看板娘-未来计划

    本次看板娘功能为公测版本。我们还在开发更多看板娘的模型与功能，将于后期陆续发布。更多细节还在调整中…正式版也将在后期发布！正式版将带来更快的模型加载时长，更多的模型种类，大部分反馈的bug也将得到修复。感谢您的耐心等待！

更多内容

  1. 搜索功能优化：本次更新后，搜索将会更快，更全面，更准确。
  2. 目录功能更新：本次更新后，目录支持更细分的小标题，小小标题。
  3. 网页整体流畅性：本次更新后，由于内容的突然增多，页面的加载速度可能有所下降。但我们同时提高了网页内容的单体加载速度，如果给您带来不便，我们深表歉意！
  4. 其他：优化了一些已知的bug。

因为是一次有着很多新内容的更新，所以网页的稳定性必然有所下降。如果您有任何意见、建议，我们都期待着您通过邮件的方式发送到ditiemi622@outlook.com，感谢您的反馈，我们将认真阅读每一篇邮件！同时，由于后期的个人问题，我们将放缓内容更新的频率并取消”周总结“的编写，我们感到非常抱歉，但是我们仍然会更新更多优质的作品，敬请期待！
